# Age Calculator

An age calculator web application that calculates your exact age based on your date of birth. It displays your age in years, months, and days accurately.

## Features
- Calculates exact age in years, months, and days
- Simple and user-friendly interface
- Accurate date calculations
- Mobile-responsive design

## Technologies Used
- HTML
- CSS
- JavaScript
